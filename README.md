# Dormitory Manager (Spring Boot)

Система управления общежитием, построенная на базе Spring Framework. Проект предназначен для отслеживания информации о студентах.

## Технологический стек

* **Runtime:** Java 21
* **Framework:** Spring Boot 4.0.3 


## Функциональные возможности

* Поиск студентов по номеру комнаты и возрасту

## Качество кода (SonarQube)

Проект проходит регулярный статический анализ кода с использованием **SonarQube** для обеспечения высоких стандартов разработки.
* **Анализ:** [Посмотреть результаты на SonarCloud](https://sonarcloud.io/project/configuration/AutoScan?id=raukouskiandrey_Dormitory-management-system)
