# Dormitory Manager (Spring Boot)

Система управления общежитием, построенная на базе Spring Framework. Проект предназначен для отслеживания информации о студентах.

## Технологический стек

* **Runtime:** Java 25
* **Framework:** Spring Boot 4.0.2
* **Maven**
* **Lombok**
* **PostgreSQL**

## Функциональные возможности

* Поиск студентов по номеру комнаты и возрасту
* Поиск студентов с помощью сложного запроса
* Добавление нарушений
* Заселение студентов в комнаты

## Качество кода (SonarQube)
